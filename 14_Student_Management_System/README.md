# Student Management System

## Project Overview

This project is a menu-driven Student Management System built using Python.

It demonstrates:

- Object-Oriented Programming (OOP)
- Inheritance
- File Handling
- Exception Handling
- Modules

## Features

- Person base class
- Student class
- Teacher class
- School class
- Add Student
- Remove Student
- Search Student
- View All Students
- Save Student Data
- Load Student Data
- Calculate Student Average
