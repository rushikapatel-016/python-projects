# Personal Utility Toolkit

A beginner-friendly Python project demonstrating the use of custom modules and built-in modules.

## Features

### Math Tools
- Area of Circle
- Area of Triangle
- Area of Rectangle

### Date Tools
- Display Today's Date
- Calculate Days Until Deadline
- Calculate Age from Birth Year

### Random Tools
- Password Generator
- Random Name Picker
- Dice Roller
