# Python Projects

## Project 1: Student Information System

### Description
This program takes a student's name, age, and city as input and displays the information in a formatted way.

### Code File
project_01_student_info.py

### Sample Output

```text
Enter your name: kush
Enter your age: 25
Enter your city: navsari

===== STUDENT INFORMATION =====

Name : kush
Age  : 25
City : navsari

==============================
```

## Project 3: Grade Checker

### Description
This program takes marks as input and displays the corresponding grade.

### Features
- Grade A (90 and above)
- Grade B (70–89)
- Grade C (50–69)
- Fail (Below 50)

### Skills Used
- Input and Output
- Comparison Operators
- Conditional Statements (if, elif, else)

### Sample Output

```text
Enter your marks: 75

===== GRADE REPORT =====
Grade: B
========================
```
